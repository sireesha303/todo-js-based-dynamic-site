# Todo Application 
* JS Based Dynamic Website

## Description
* Dynamic website Frontend built with HTML5, CSS3 and Bootstrap and JavaScript.
* User can add todo, mark the same as completed and can delete.

## Technologies Used
* HTML5
* CSS3
* Bootstrap 4
* JavaScript
